# 工作流编辑器 V2 - 完整功能文档

## 访问方式

- 侧边栏「工作流」菜单
- 首页「工作流编辑器」卡片
- 直接访问：http://localhost:5000/workflow

## 节点类型（10种）

### 基础节点
1. **文本输入** - 输入文本内容
2. **图片输入** - 输入图片 URL

### 图像生成节点
3. **文生图** - 文本生成图片
   - 参数：提示词、负面提示词、宽度、高度、风格、随机种子、迭代步数、引导强度
4. **图生图** - 图片修改生成
   - 参数：修改提示词、修改强度、随机种子、迭代步数、引导强度

### 视频生成节点
5. **图生视频** - 图片生成视频
   - 参数：动作描述、时长、比例、运动强度
6. **文生视频** - 文本生成视频
   - 参数：场景描述、时长、比例

### 人物生成节点
7. **文生人物** - 文字生成人物形象
   - 参数：人物描述、风格
8. **人物三视图** - 生成人物正侧背三视图
   - 参数：风格

### 语音生成节点
9. **文字转语音** - 文本转语音
   - 参数：要转换的文本、音色、语速

### LLM 处理节点
10. **LLM 处理** - 大语言模型处理
    - 参数：处理指令、模型

## 使用示例

### 示例 1：文生图工作流

1. 添加「文本输入」节点
2. 输入：`一只可爱的橘猫坐在窗边，阳光洒进来，写实风格`
3. 添加「文生图」节点
4. 连接：文本输入.text → 文生图.prompt
5. 设置参数：
   - 宽度：1024
   - 高度：1024
   - 风格：realistic
   - 迭代步数：30
6. 点击「执行工作流」
7. 查看生成的图片

### 示例 2：图生视频工作流

1. 添加「图片输入」节点
2. 输入图片 URL
3. 添加「图生视频」节点
4. 连接：图片输入.image → 图生视频.referenceImage
5. 设置参数：
   - 动作描述：`猫咪缓慢地转过头看向镜头`
   - 时长：5秒
   - 比例：16:9
   - 运动强度：5
6. 点击「执行工作流」
7. 查看生成的视频

### 示例 3：人物三视图工作流

1. 添加「图片输入」节点
2. 输入人物参考图 URL
3. 添加「人物三视图」节点
4. 连接：图片输入.image → 人物三视图.referenceImage
5. 设置参数：
   - 风格：realistic
6. 点击「执行工作流」
7. 查看生成的三视图（正、侧、背）

### 示例 4：复杂工作流

```
文本输入 → LLM 处理 → 文生图 → 图生视频
```

1. 添加节点：文本输入 → LLM 处理 → 文生图 → 图生视频
2. 按顺序连接所有节点
3. 配置每个节点的参数
4. 点击「执行工作流」

### 示例 5：使用模板

1. 点击右侧「AI」标签
2. 在「快捷模板」中选择一个模板
3. 点击应用
4. 修改参数
5. 执行工作流

### 示例 6：保存和加载工作流

**保存工作流：**
1. 配置好工作流
2. 点击顶部「保存」按钮
3. 输入工作流名称
4. 点击「保存」

**加载工作流：**
1. 点击右侧「AI」标签
2. 在「保存的工作流」中选择一个工作流
3. 点击播放按钮加载

## API 复用说明

工作流编辑器复用了以下现有 API：

1. `/api/generate/scene-image` - 文生图
2. `/api/generate/appearance-from-image` - 图生图
3. `/api/generate/videos` - 视频生成
4. `/api/generate/appearance-from-text` - 文生人物
5. `/api/generate/character-triple-views` - 三视图生成
6. `/api/generate/voice` - 语音生成
7. 存储路径：统一使用阿里云 OSS / S3

## 存储逻辑

- 节点参数：自动保存在内存中
- 工作流：保存在 localStorage
- 生成的文件：保存在 OSS/S3
- 自动获取公网 URL 供访问

## 故障排除

### 问题：节点无法连接
- 确保点击的是输出端口（右侧圆点）
- 拖拽到输入端口（左侧圆点）
- 松开鼠标完成连接

### 问题：参数面板不显示
- 确保选中了节点（节点周围有高亮边框）
- 点击节点即可选中

### 问题：执行失败
- 检查节点是否正确连接
- 检查参数是否填写完整
- 查看节点状态（红色表示失败）

### 问题：预览不显示结果
- 确保工作流已执行
- 选中节点查看预览
- 检查节点执行状态
