# 工作流编辑器 V2 - 使用说明

## 概述

工作流编辑器 V2 是一个参考即梦画布设计的节点式可视化编辑器，支持拖拽式节点编排、实时参数配置和一键执行。

## 核心功能

### 1. 节点库（左侧面板）

提供以下节点类型：

- **文本输入** - 输入文本内容
- **图片输入** - 输入图片 URL
- **文生图** - 文本生成图片
- **图生图** - 图片修改生成
- **图生视频** - 图片生成视频
- **文生视频** - 文本生成视频
- **LLM 处理** - 大语言模型处理

### 2. 画布区域（中间）

- **拖拽添加节点**：从左侧节点库拖拽节点到画布
- **节点移动**：拖拽节点可调整位置
- **节点连接**：
  1. 点击节点的输出端口（右侧圆点）
  2. 拖拽到另一个节点的输入端口（左侧圆点）
  3. 松开鼠标完成连接
- **缩放和平移**：
  - 使用鼠标滚轮缩放
  - 按住 Alt 键 + 鼠标左键拖拽平移
- **删除连接**：点击连接线删除
- **删除节点**：点击节点右上角删除按钮

### 3. 参数面板（右侧 - 参数标签）

- **选择节点**：点击画布上的节点，右侧显示该节点的参数
- **参数类型**：
  - **文本框** - 单行文本输入
  - **多行文本** - 多行文本输入（如提示词）
  - **数字** - 数值输入（如随机种子）
  - **下拉框** - 选项选择（如风格、比例）
  - **滑块** - 数值调节（如迭代步数、引导强度）
- **实时修改**：修改参数后自动保存

### 4. 预览面板（右侧 - 预览标签）

- **执行结果**：显示节点执行后的输出结果
- **图片预览**：显示生成的图片
- **视频预览**：显示生成的视频
- **文本输出**：显示文本结果

### 5. AI 面板（右侧 - AI 标签）

- **智能生成**（开发中）：通过自然语言描述自动生成工作流
- **快捷模板**（开发中）：预设常用工作流模板

## 使用流程

### 示例 1：文生图工作流

1. 添加「文本输入」节点
2. 在参数面板输入提示词
3. 添加「文生图」节点
4. 将「文本输入」的输出连接到「文生图」的 prompt 输入
5. 在「文生图」节点设置参数（宽度、高度、风格等）
6. 点击顶部「执行工作流」按钮
7. 在预览面板查看生成的图片

### 示例 2：图生视频工作流

1. 添加「图片输入」节点
2. 在参数面板输入图片 URL
3. 添加「图生视频」节点
4. 将「图片输入」的输出连接到「图生视频」的 referenceImage 输入
5. 在「图生视频」节点设置参数（动作描述、时长、比例等）
6. 点击「执行工作流」
7. 在预览面板查看生成的视频

### 示例 3：复杂工作流

可以组合多个节点创建复杂工作流：

```
文本输入 → LLM 处理 → 文生图 → 图生视频
```

## 节点类型详解

### 文本输入节点

- **参数**：
  - `text` (文本框) - 输入的文本内容
- **输出**：
  - `text` - 文本内容

### 图片输入节点

- **参数**：
  - `imageUrl` (文本框) - 图片 URL
- **输出**：
  - `image` - 图片 URL

### 文生图节点

- **输入**：
  - `prompt` (字符串) - 提示词
- **参数**：
  - `prompt` (多行文本) - 提示词
  - `negativePrompt` (多行文本) - 负面提示词
  - `width` (下拉框) - 宽度 (512/768/1024/1360)
  - `height` (下拉框) - 高度 (512/768/1024/1360)
  - `style` (下拉框) - 风格 (realistic/anime/cartoon/oil_painting/watercolor)
  - `seed` (数字) - 随机种子 (-1 为随机)
  - `steps` (滑块) - 迭代步数 (10-50)
  - `guidance` (滑块) - 引导强度 (1-20)
- **输出**：
  - `image` - 生成的图片 URL

### 图生图节点

- **输入**：
  - `referenceImage` (图片) - 参考图片
  - `prompt` (字符串) - 修改提示词
- **参数**：
  - `prompt` (多行文本) - 修改提示词
  - `strength` (滑块) - 修改强度 (0-1)
  - `seed` (数字) - 随机种子
  - `steps` (滑块) - 迭代步数
  - `guidance` (滑块) - 引导强度
- **输出**：
  - `image` - 生成的图片 URL

### 图生视频节点

- **输入**：
  - `referenceImage` (图片) - 参考图片
  - `motionPrompt` (字符串) - 动作描述
- **参数**：
  - `motionPrompt` (多行文本) - 动作描述
  - `duration` (滑块) - 时长 (4-12 秒)
  - `aspectRatio` (下拉框) - 比例 (16:9/9:16/1:1)
  - `motionStrength` (滑块) - 运动强度 (1-10)
- **输出**：
  - `video` - 生成的视频 URL

### 文生视频节点

- **输入**：
  - `prompt` (字符串) - 场景描述
- **参数**：
  - `prompt` (多行文本) - 场景描述
  - `duration` (滑块) - 时长 (4-12 秒)
  - `aspectRatio` (下拉框) - 比例 (16:9/9:16/1:1)
- **输出**：
  - `video` - 生成的视频 URL

### LLM 处理节点

- **输入**：
  - `input` (字符串) - 输入文本
- **参数**：
  - `prompt` (多行文本) - 处理指令
  - `model` (下拉框) - 模型 (deepseek-chat/gpt-4/claude-3)
- **输出**：
  - `output` - 处理结果

## 执行机制

1. **拓扑排序**：自动根据节点依赖关系确定执行顺序
2. **并行执行**：无依赖的节点并行执行，提高效率
3. **数据传递**：通过连接线自动传递数据
4. **状态管理**：
   - 运行中：节点闪烁
   - 成功：绿色图标
   - 失败：红色图标
5. **错误处理**：单个节点失败不影响其他节点

## 快捷键

- `Alt + 鼠标左键拖拽` - 平移画布
- `鼠标滚轮` - 缩放画布
- `点击节点` - 选中节点，查看参数
- `点击连接线` - 删除连接

- **UI 组件**：shadcn/ui
- **状态管理**：React Hooks
- **样式**：Tailwind CSS 4
