# 场景规则：氛围短片 (atmosphere_short)

> **所属 Skill**: gen-video
> **scene-rule ID**: atmosphere_short
> **核心策略**: 沉浸式情绪叙事
> **适用场景**: 情感表达优先的氛围短片、治愈系内容、情绪叙事

---

## 🎯 核心特征

**氛围短片的核心目标是创造沉浸式的情绪体验**，而非传递信息或讲述复杂情节。画面、音乐、节奏共同服务于情绪氛围的营造。

---

## 📋 规则详情

| 维度 | 规则 |
|------|------|
| **分镜设计** | 频繁使用前景物理遮挡，增强画面层次感和沉浸感 |
| **氛围遮挡** | 使用水滴、烟雾、光影等氛围元素作为前景，营造朦胧美感 |
| **运镜** | 默认固定镜头，或小幅度晃动（模拟呼吸感），避免大幅度快速运镜 |
| **动作描述** | 按时序拆解动作，通过肢体语言传达情绪，避免抽象情绪词 |
| **文案风格** | 不使用"画面解说"。多用第三人称叙事或内心独白 |
| **分时序密度** | 默认 4-5s/段（慢节奏），给观众情绪消化空间 |
| **旁白处理** | 默认无旁白或少量旁白。有旁白时必须用 `旁白说："台词"` 格式 |
| **BGM** | 总时长 ≥15s 时 ✅ 必须；总时长 <15s 时可选 |
| **音色管线** | 🅰️ **管线 A（描述音色）** |

---

## 🎬 分镜设计要点

### 1. 前景遮挡增强沉浸感

**常用前景元素**：
- 自然元素：树叶、花朵、雨滴、雪花、水雾
- 人造元素：窗框、栏杆、门框、纱帘
- 光影元素：光斑、阴影、逆光剪影

**示例 Prompt**：
```
0-5s：【全景】透过半透明的纱帘，场景[图片1]在晨雾中若隐若现。角色[图片2]站在窗前，背对镜头。
5-10s：【中景】镜头缓慢推进，纱帘在微风中轻摆，光影斑驳地洒在角色身上。
不添加字幕，不添加BGM
```

### 2. 氛围元素作为前景

**氛围元素类型**：
- 水滴/雨丝：营造忧郁、清新、怀旧感
- 烟雾/晨雾：营造朦胧、梦幻、神秘感
- 光尘/光斑：营造温暖、希望、神圣感
- 花瓣/落叶：营造浪漫、感伤、时光流逝感

**示例 Prompt**：
```
0-6s：【近景】雨滴打在玻璃窗上，模糊了窗外的街景。角色[图片1]的手指轻轻划过玻璃，留下水痕。
6-12s：【特写】镜头聚焦在玻璃上的水痕，水痕慢慢滑落。
不添加字幕，不添加BGM
```

### 3. 固定镜头 + 微晃动

**运镜建议**：
- 默认：固定镜头（镜头静止，画面稳定）
- 可选：小幅度晃动（模拟呼吸感，增强纪实/亲密感）
- 避免：大幅度快速运镜（破坏氛围沉浸感）

**示例 Prompt**：
```
0-8s：【中景】镜头微微晃动，仿佛观察者的呼吸。角色[图片1]坐在桌前，双手捧着茶杯，目光低垂。
不添加字幕，不添加BGM
```

## 💬 文案风格要点

### 1. 禁止"画面解说"

❌ **错误示例**：
```
画面中是一个年轻女子坐在窗边，窗外下着雨，她的表情有些忧郁。
```

✅ **正确示例**：
```
0-5s：【中景】角色[图片1]坐在窗边，手指无意识地摩挲着茶杯边缘。窗外的雨丝在玻璃上蜿蜒而下。
5-10s：【近景】角色微微抬头，眼神望向窗外，嘴角轻轻抿起。
不添加字幕，不添加BGM
```

### 2. 第三人称叙事

**适用场景**：无旁白，通过画面和动作传达故事。

**示例**：
```
0-4s：【全景】场景[图片1]的街道在暮色中延伸。角色[图片2]独自走在街道中央，脚步缓慢。
4-8s：【中景】角色停下脚步，回头望向来时的方向，眼神中带着一丝不舍。
8-12s：【全景】角色转身继续前行，身影渐渐消失在暮色中。
不添加字幕，不添加BGM
```

### 3. 内心独白

**适用场景**：有旁白，通过内心独白传达角色情绪。

**示例**：
```
0-5s：【近景】角色[图片1]站在海边，海风吹动她的长发。旁白说："有时候，离开是为了更好的相遇。"
5-10s：【中景】角色脱下鞋子，赤脚踩在沙滩上，一步一步走向海浪。
不添加字幕，不添加BGM
```

## 🎵 BGM 配乐建议

**情绪类型 → BGM 风格对应**：

| 情绪基调 | BGM 风格 | gen_music 示例 |
|---------|---------|---------------|
| 治愈温暖 | 轻柔钢琴/弦乐 | 缓慢的环境氛围纯音乐，钢琴+弦乐，温暖朦胧 |
| 怀旧感伤 | 木吉他/口琴 | 缓慢的民谣纯音乐，木吉他独奏，怀旧感伤 |
| 神秘朦胧 | 合成器/环境音 | 环境氛围纯音乐，合成器pad+自然采样，朦胧神秘 |
| 浪漫甜蜜 | 轻快钢琴/吉他 | 轻快的流行纯音乐，钢琴+吉他，浪漫甜蜜 |
| 忧郁孤独 | 大提琴/钢琴 | 缓慢的古典纯音乐，大提琴独奏，忧郁孤独 |

## ⏱️ 分时序密度建议

**氛围短片默认慢节奏**，给观众情绪消化空间。

| 动作类型 | 建议时长 | 原因 |
|---------|---------|------|
| 空镜/纯氛围 | 5-6s | 给观众沉浸时间 |
| 单一动作（转身、抬手） | 4-5s | 动作缓慢，情绪饱满 |
| 连续动作链 | 3-4s/段 | 动作流畅，节奏均匀 |
| 情绪高潮 | 3-4s/段 | 节奏稍快，增强冲击 |

**示例**：
```
0-5s：【全景】场景[图片1]的湖面在晨雾中静谧如镜。一只白鹭掠过水面，激起涟漪。
5-10s：【中景】角色[图片2]站在湖边，双手插在口袋里，静静地看着远方。
10-15s：【特写】镜头推近角色的侧脸，风吹动她的发丝，她的嘴角轻轻上扬。
不添加字幕，不添加BGM
```

## 📝 完整示例

### 场景：春日晨曦中的小镇

**剧本**：
- 陈念在春日清晨走出家门，漫步在小镇街道上，感受春天的气息。
- 情绪：宁静、治愈、希望

**分镜设计**：

**Shot 1**（8s）：
```
0-4s：【全景】透过桃花枝条的前景，场景[图片1]的小镇在晨雾中苏醒。炊烟袅袅升起。
4-8s：【中景】镜头缓慢推进，晨光穿过薄雾，洒在青石板路上。
不添加字幕，不添加BGM
```

**Shot 2**（10s）：
```
0-5s：【中景】角色[图片2]推开木门，走出小屋，伸了个懒腰。晨光映在他的脸上。
5-10s：【全景】角色沿街道缓步前行，两旁是白墙黑瓦的民居，桃花在风中轻摆。
不添加字幕，不添加BGM
```

**Shot 3**（8s）：
```
0-4s：【近景】角色停在溪边，蹲下身，手指轻轻触碰水面。涟漪散开。
4-8s：【特写】镜头聚焦在水面的涟漪，倒映出蓝天和桃花。
不添加字幕，不添加BGM
```

**BGM**：

gen_music(

## ⛔ 常见错误与规避

### 错误1：信息量过载

❌ **错误做法**：
- 一个 10s 的 shot 塞入 3 个情节点
- 镜头快速切换，破坏氛围

✅ **正确做法**：
- 一个 shot 只传达一个核心情绪
- 镜头切换缓慢，给观众情绪过渡时间

---

### 错误2：使用"画面解说"式旁白

❌ **错误示例**：
```
画面中是一个年轻男子走在街道上，街道两旁是古老的建筑。
```

✅ **正确示例**：
```
0-5s：【全景】角色[图片1]独自走在青石板路上，两旁的白墙黑瓦在晨光中静默伫立。
不添加字幕，不添加BGM
```

---

### 错误3：运镜过于复杂

❌ **错误做法**：
```
0-5s：【全景→中景→特写】镜头快速推进，从全景直切到面部特写。
```

✅ **正确做法**：
```
0-5s：【全景】镜头固定，画面稳定。
5-10s：【中景】镜头缓慢推进，节奏舒缓。
```
