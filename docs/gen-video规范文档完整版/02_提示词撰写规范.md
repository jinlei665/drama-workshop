# 提示词撰写规范（V3 设定稿 + V5 分镜融合法则）

> **所属 Skill**: gen-video
> **核心用途**: 规范从静态参考图（V3）到动态视频分镜（V5）的 Prompt 语法
> **何时读取**: V3 生成参考图时必读；V5 阶段如需确认 `[图片N]` 融合语法时可参考

---

## 🌐 全局写作原则（V3/V5 通用）

**原则 1：结构化描述，禁止自由发挥**
不要加修饰语、比喻、人名引用。槽位只填**可视化的物理特征**。
- ✗ `王家卫电影中那种让人过目不忘的女孩`
- ✓ `圆润脸型，黑色利落短发齐耳，大而明亮的眼睛，浅蓝色宽松衬衫`

**原则 2：构图指令必须冗余**
关键构图参数需要多说一遍，用约束条件加固（正面 → 加 `面向镜头，五官清晰可见`）。

**原则 3：具象化描述，禁止泛化和歧义**
- ✗ `像一只都市里的飞鸟`（比喻）→ ✓ `微微歪头，整体气质自由而率性`
- ✗ `深空场景`（泛化）→ ✓ `深邃宇宙深空，远处黑洞吸积盘金色弧光环绕，光线被引力场弯曲`

**原则 4：文字内容必须显式指定**
- ✗ `货架上方标注着标签`（模型不知道写什么字）→ ✓ `罐头正面标签上印着"2024.05"`

**原则 5：变量替换铁律**
公式中的 `[...]` 必须替换为具体描述！**最终 Prompt 中绝不能输出 `[]` 符号（V5 的 `[图片N]` 引用除外）。**

## 🟢 第一部分：V3 设定稿生图公式（⚠️ 仅限 V3 阶段）

**⛔ V3 核心法则：分离资产！角色不带环境，场景不带人物。**
*所有 V3 Prompt 必须以 `[unified_style_prompt]` 的实际内容作为开头。*

### 1. 角色类 Prompt（建立人物锚点）

只画角色本体，不画环境不画手持物。角色与道具必须分离生图（出现 ≥2 镜头的物品单独提取）。

**公式（横屏三视图设定稿）**：

```
16:9 横版构图，[unified_style_prompt]。
左半边区域居中展示角色面部特写，
右半边区域整齐排布角色正面、侧面及背面三张全身设定图。
纯白背景，无文字无水印。
[①体型：性别+年龄段+身高+体型]，[②面孔：东方人/西方人面孔]，
[③气质：整体气质关键词，日常神态]，
[④发型：发色+发长+发式+质感]，
[⑤面部：脸型+皮肤+五官特征]，
[⑥服装主体：款式+色彩+材质]，
[⑦服装细节：配饰+鞋靴]，
[⑧身体标记：伤疤/纹身/胎记（如有）]，
[⑨姿态：站姿描述]
```

**有参考图时**：`16:9 横版构图，[unified_style_prompt]。左半边面部特写，右半边正面/侧面/背面三张全身设定图。保持原参考图主体特征。纯白背景，无文字无水印。`

**注意**：
- **⛔ 角色设定稿固定 16:9 横版**，不受视频画幅（9:16竖屏等）影响——设定稿是参考素材不是视频帧
- ①-⑨按剧本取舍，有设定的必写，无设定可省
- 动物角色用体型描述，严禁拟人化（除非剧本要求）
- **⛔ 禁止混入环境描述**：角色图 = 白色背景 + 角色本体。环境描述属于场景图。

### 2. 场景类 Prompt（建立空间锚点）

只画环境，不画人物。

**公式（分层描述）**：

```
[unified_style_prompt]，
画面中不出现任何角色；纯场景、纯环境、无人物，
[①视角：从哪里看向哪里+景别]，
[②空间类型：室内/室外+时代/世界观]，
[③前景：距镜头最近的物体]，
[④中景：场景主体结构]，
[⑤远景：天际线/背景/纵深尽头]，
[⑥关键物件：有叙事功能的特定物体]，
[⑦光照：光源方向+类型+强度]，
[⑧色调与氛围：主色调+情绪关键词]，
画面干净无文字无logo，[比例]
```

### 3. 道具类 Prompt（武器/法器/信物等）

```
[unified_style_prompt]，
[物件类型+名称]，纯白背景，正面视角，
[材质+表面质感]，[形状+颜色+尺寸参照]，[细节/痕迹]，
光照均匀，无文字无水印，[比例]
```

### 3.5 产品摄影 Prompt（电商/品牌宣传专属）

```
[①摄影技法]，[②光影设计]，
[③产品主体+材质精描]，
[④构图+视角+角度]，
[⑤色调美学]，
纯白背景 或 [指定色系背景]，
无文字无水印，[比例]
```

| 层级 | 美妆/香水 | 3C/科技 | 食品/饮品 | 汽车 |
|------|----------|---------|----------|------|
| ①摄影技法 | 柔焦质感，慢门摄影 | 微距摄影，锐利对焦 | 暗调摄影，蒸汽氤氲 | 影棚专业布光 |
| ②光影设计 | 漫射柔光，朦胧光晕 | 侧向硬光，金属反射高光 | 暖色侧光，食欲光感 | 戏剧性明暗对比 |
| ③材质精描 | 磨砂/透明玻璃/液体质感 | 阳极氧化铝/镜面不锈钢 | 水珠凝结/奶油流淌 | 哑光车漆/镀铬金属 |
| ④构图视角 | 低角度仰拍 | 45度俯视，几何排列 | 平视特写，浅景深 | 正侧45度，低角度 |
| ⑤色调美学 | 低饱和冷调，杂志级调色 | 深空灰冷调，科技蓝 | 暖调暗调，高饱和 | 影棚中性灰，金属冷调 |

### 4. 音色描述公式（`--voice` 入库时使用）

`[性别]声，[年龄段]音色，音调[特征]，音色质感[质感]，发音方式[特征]，带有[情感气质]`

## 🔵 第二部分：V5 分镜 Prompt 融合法则（⚠️ 仅限 V5 多素材模式）

**⛔ V5 核心法则：动态融合！打破 V3 的分离规则，将角色放入场景并赋予动作。**

*若当前为首尾帧模式，请直接遵循 `firstframe-guide.md`，忽略此部分。*
*V5 的 JSON 结构与分时序语法（`0-4s：【景别】...`）请严格遵循 `shot-script-guide-core.md`。此处仅规范画面描述的内容逻辑。*

### 1. 打破白底与空镜限制
在 V5 的 `aigc_prompt` 中，**必须忘记** V3 的"纯白背景"、"无人物"规则。V5 的目标是**把独立的素材融合进同一个画面**。

### 2. `[图片N]` 引用与空间定位铁律
必须通过明确的介词（在...中 / 站在...旁 / 位于...）将角色图与场景图在画面中锚定。

**正确**：`0-4s：【近景】[图片1] 站在 [图片2] 的霓虹灯牌下，微微低头，眼神警惕地看向左侧。`
**错误**：`0-4s：【近景】男主站在街上。`（没有 [图片N] 引用 → 引擎随机生成路人）

**防幻觉铁律**：Prompt 中提及的任何核心人物或重要道具，**必须**挂载对应的 `[图片N]`。不加引用 = 不连戏。

### 3. 动态与物理交互
视频 Prompt 必须包含随时间变化的**具体视觉动作**。
- **正确（物理动作）**：`[图片1] 抬起右手遮挡刺眼阳光，微风吹动头发。`
- **错误（文学描述）**：`[图片1] 陷入了沉思`（模型无法渲染抽象心理状态）
- **错误（超自然瞬移）**：`[图片1] 化作一阵烟雾消失`（非连续动作无法生成）
