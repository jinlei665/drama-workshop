# 场景规则：预告片/Trailer (trailer_preview)

> **所属 Skill**: gen-video
> **scene-rule ID**: trailer_preview
> **核心策略**: 高密度分时序快切 + 多场景色系对比
> **适用场景**: 电影预告片、剧集预告、游戏预告、品牌宣传片

---

## 🎯 核心认知

**预告片的核心是"钩子+悬念"——用最短的时间抓住观众注意力，留下强烈的观看欲望。**

| ❌ 错误做法 | ✅ 正确做法 |
|------------|-----------|
| 15 个 4s 的 shot | 5-7 个 10-12s 的 shot，内部 2-3s/段密集分时序 |
| 同一场景拆成多个 shot | 同一场景 1 个 shot，多段快切 |

---

## 📋 规则详情

| 维度 | 规则 |
|------|------|
| **叙事结构** | 五段式：① 钩子开场 → ② 角色亮相 → ③ 冲突/对抗 → ④ 转折或情感高潮 → ⑤ 标题卡+悬念 |
| **场景数量** | 至少 **3 个**视觉差异明显的**纯环境**场景参考图 |
| **分时序密度** | **全片默认 2-3s/段** |
| **shot 时长** | 大部分 shot 10-12s |
| **台词与旁白** | 全片总台词 ≤ 30 字。每句 ≤ 10 字 |
| **BGM** | ⛔ **BGM 前置**。必须在 V2.5 阶段完成选曲/生曲 + analyze_bgm |
| **音色管线** | 🅰️ **管线 A（描述音色）** |

## 🎬 五段式叙事结构

### 第一段：钩子开场（前 3-5 秒）

**目标**：用最具冲击力的画面/音效抓住观众。

**常用手法**：
- 视觉冲击：爆炸、崩塌、超现实画面
- 听觉冲击：巨响、寂静后的音效
- 情绪冲击：恐惧、震撼、好奇

**示例**：
```
Shot 1（12s）：
0-3s：【全景】场景[图片1]的城市在夜幕中突然崩塌，碎片飞溅。
3-6s：【中景】建筑倒塌，烟尘四起。
6-9s：【特写】一只眼睛在烟尘中睁开，眼神锐利。
9-12s：【全景】城市废墟中，一个身影屹立不倒。
不添加字幕，不添加BGM
```

### 第二段：角色亮相（5-15 秒）

**目标**：展示主角形象，建立角色认知。

**常用手法**：
- 正面特写：展示主角面容
- 动作亮相：主角的关键动作（拔剑、转身、抬头）
- 对白亮相：主角的第一句台词（≤ 10 字）

**示例**：
```
Shot 2（10s）：
0-3s：【中景】角色[图片2]站在废墟中，背对镜头，风吹动他的衣摆。
3-6s：【近景】角色缓缓转身，眼神坚定。角色说："我会找到答案。"
6-10s：【全景】镜头拉远，角色独自站在废墟中，远处是残破的天际线。
不添加字幕，不添加BGM
```

### 第三段：冲突/对抗（15-35 秒）

**目标**：展示核心冲突，营造紧张氛围。

**常用手法**：
- 快切战斗：2-3s/段的密集分时序
- 对峙场面：主角 vs 反派
- 危机展现：灾难、追逐、困境

**示例**：
```
Shot 3（12s）：
0-3s：【中景】角色[图片2]面对场景[图片3]中的黑暗势力，剑拔弩张。
3-6s：【近景】反派角色[图片4]冷笑，眼神阴冷。反派角色说："你逃不掉。"
6-9s：【中景】主角握紧武器，准备迎战。
9-12s：【全景】双方对峙，气氛紧张到极点。
不添加字幕，不添加BGM
```

### 第四段：转折或情感高潮（35-50 秒）

**目标**：制造情绪冲击，升华主题。

**常用手法**：
- 情感爆发：角色的情绪崩溃/觉醒
- 视觉奇观：特效场面、大场景
- 音乐高潮：配合 BGM 的高潮段落

**示例**：
```
Shot 4（10s）：
0-3s：【特写】角色[图片2]的眼睛涌起光芒，力量觉醒。
3-6s：【全景】角色爆发出强大的能量，周围的空间被撕裂。
6-10s：【中景】角色抬头，眼神中带着决绝。角色说："这次，我不会输。"
不添加字幕，不添加BGM
```

### 第五段：标题卡+悬念（最后 5-10 秒）

**目标**：留下悬念，强化观看欲望。

**常用手法**：
- 标题卡：黑屏+剧名
- 悬念台词：一句勾起好奇的台词
- 日期卡：上映日期

**示例**：
```
Shot 5（8s）：
0-3s：【全景】角色[图片2]的背影消失在光芒中。
3-6s：【黑屏】画面渐黑，标题浮现："《命运之刃》"。
6-8s：【黑屏】副标题："2024 年，命运即将揭晓。"
不添加字幕，不添加BGM
```

## 🎵 BGM 前置流程（V2.5）

**预告片必须 BGM 前置**，因为分镜节奏受音乐节拍驱动。

### Step 1：选曲或生曲

```python
# 生成定制预告片配乐
gen_music(
    style="120BPM，史诗恢弘的东方管弦纯音乐，古筝琶音+弦乐群渐强+战鼓，空间宏大深远",
    music_name="BGM_预告片",
    is_instrumental=True,
    video_script_file="命运之刃_预告片_视频脚本.cvs"
)
```

### Step 2：节拍分析

```bash
python gen-video/scripts/analyze_bgm.py --bpm 120 --duration-ms 60000
```

### Step 3：读取 beat_map.json

beat_map.json 会生成：
- `beats`：每个节拍的时间点
- `sections`：音乐段落（Intro/Build Up/Chorus/Outro）
- `recommended_shot_cuts`：推荐的 shot 切换点

### Step 4：按节拍写分镜

**音乐驱动模式**：
- shot 边界：从 `recommended_shot_cuts` 选取
- 分时序段落：每个 beat 对应一个段落
- 视觉强度：根据 `sections.energy` 调整

## 💬 台词与旁白控制

**核心规则**：
- 全片总台词 ≤ 30 字
- 每句 ≤ 10 字

**原因**：预告片的核心是视觉冲击，台词过多会分散注意力。

**示例**：
```
角色说："我会找到答案。"（7 字）
反派角色说："你逃不掉。"（4 字）
角色说："这次，我不会输。"（7 字）
```

**总字数**：7 + 4 + 7 = 18 字，符合 ≤ 30 字要求。

## ⛔ 常见错误与规避

### 错误1：拆成过多 shot

❌ **错误做法**：
- 60s 预告片，拆成 15 个 4s 的 shot
- 每个场景一个 shot

✅ **正确做法**：
- 60s 预告片，5-7 个 10-12s 的 shot
- 同一场景用分时序快切，不拆 shot

---

### 错误2：台词过多

❌ **错误做法**：
- 全片台词 50+ 字
- 每句台词 15+ 字

✅ **正确做法**：
- 全片台词 ≤ 30 字
- 每句 ≤ 10 字

---

### 错误3：跳过 BGM 前置

❌ **错误做法**：
- V2.5 不执行，直接进 V3
- V6.5 才选曲

✅ **正确做法**：
- V2.5 必须执行：选曲 → 节拍分析 → 写入 bgm_list
- V5 写分镜时按节拍切分
