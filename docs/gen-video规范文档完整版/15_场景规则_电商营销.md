# 场景规则：电商营销 (ecommerce_marketing)

> **所属 Skill**: gen-video
> **scene-rule ID**: ecommerce_marketing
> **核心策略**: 产品质感展示与物理逻辑自洽
> **适用场景**: 产品广告、电商带货、品牌宣传片

---

## 🎯 核心特征

**电商营销的核心目标是激发购买欲望**。画面设计必须聚焦产品质感、使用场景、核心卖点，通过精准的视觉呈现和有说服力的旁白来推动转化。

---

## 📋 规则详情

| 维度 | 规则 |
|------|------|
| **节奏设计** | 三段式节奏：前3秒钩子 → 中段至少3个卖点递进 → 结尾Slogan |
| **文案设计** | 产品卖点必须用旁白说明，禁止复杂说明和抽象描述 |
| **旁白处理** | 电商广告的旁白是画外音（VO），V3 阶段必须创建旁白虚拟角色 |
| **音画同步** | 旁白与画面必须一一对应 |
| **运镜设计** | 镜头选择以服务信息点为主，动态运镜只允许单轴运动 |
| **音频设计** | 放大与产品相关的声音；分镜时长 = 旁白预估时长 + 1s 留白 |
| **时长策略** | 优先用长 shot + 分时序，不要拆短 shot |
| **场景素材** | 电商视频通常涉及 2-3 个使用场景，必须在 V3/V4 阶段为每个场景生成参考图 |
| **BGM** | ✅ 推荐。用 `select_music`，风格轻快明亮 |
| **音色管线** | 🅱️ **管线 B（预制音色）** |
| **产品与场景分离** | 产品图和场景图**必须分别生成** |

---

## 🎬 三段式节奏设计

### 第一段：前3秒钩子

**目标**：用最具吸引力的画面/信息抓住观众。

**常用手法**：
- 视觉冲击：产品特写、质感细节
- 痛点切入："你是否也在为...困扰？"
- 效果展示：使用前后对比

**示例**：
```
Shot 1（8s）：
0-3s：【特写】产品[图片1]在柔和光线下旋转，表面光泽流转。旁白说："这款面霜，让你的肌肤焕发新生。"
3-6s：【中景】产品[图片1]放在木质台面上，背景是清晨的窗景。旁白说："天然成分，温和不刺激。"
6-8s：【特写】镜头聚焦产品质地，质地细腻。旁白说："一抹吸收，清爽不油腻。"
不添加字幕，不添加BGM
```

### 第二段：中段卖点递进（至少3个卖点）

**目标**：清晰展示产品核心卖点，建立信任感。

**常用手法**：
- 功能卖点：产品功能逐一展示
- 场景卖点：不同使用场景切换
- 对比卖点：与竞品/旧款对比

**示例**：
```
Shot 2（12s）：
0-4s：【中景】场景[图片2]（卧室），产品[图片1]放在床头柜上。旁白说："睡前使用，第二天肌肤水润饱满。"
4-8s：【中景】场景[图片3]（办公室），产品[图片1]放在办公桌上。旁白说："午休补涂，瞬间提亮肤色。"
8-12s：【特写】镜头聚焦产品成分表。旁白说："富含透明质酸、烟酰胺，科学配比，效果看得见。"
不添加字幕，不添加BGM
```

### 第三段：结尾Slogan（最后 3-5 秒）

**目标**：强化品牌记忆，提供行动指引。

**常用手法**：
- 品牌Slogan："XXX，让美丽更简单"
- 行动号召："现在下单，立享优惠"
- 品牌露出：品牌Logo + 产品全景

**示例**：
```
Shot 3（5s）：
0-3s：【全景】产品[图片1]居中，背景是品牌色渐变。旁白说："XXX面霜，让美丽更简单。"
3-5s：【特写】品牌Logo浮现。旁白说："限时优惠，点击下方链接。"
不添加字幕，不添加BGM
```

## 📦 产品视频素材生成规程

### 前置：产品信息核实

**核心规则**：用户提到了具体产品名时，先用 `search_web` 搜索确认。

**示例**：
```python
search_web(query_list=["iPhone 15 Pro", "产品参数", "外观设计"])
```

### 第一步：产品主体

**工具**：`text_to_images`

**Prompt 公式**（产品摄影）：
```
[①摄影技法]，[②光影设计]，
[③产品主体+材质精描]，
[④构图+视角+角度]，
[⑤色调美学]，
纯白背景 或 [指定色系背景]，
无文字无水印，[比例]
```

**示例（美妆产品）**：
```
柔焦质感，慢门摄影，
漫射柔光，朦胧光晕，
磨砂玻璃瓶身，乳白色液体透过瓶身可见，金色瓶盖，正面标签印着"Hydra Boost"，
低角度仰拍，瓶身居中，
低饱和冷调，杂志级调色，
纯白背景，
无文字无水印，16:9
```

**入库命令**：
```bash
python gen-video/scripts/update_material_files.py "./素材库.json" \
  --item "面霜" "https://产品图URL/" "产品主视图"
```

### 第二步：场景环境

**核心规则**：根据视频内容规划 2-3 个使用场景。

**示例（面霜产品）**：

**场景1：卧室（睡前使用）**：
```
电商产品摄影风格，画面中不出现任何角色；纯场景、纯环境、无人物，
全景视角，平视，
室内，卧室，夜晚，
前景：木质床头柜，台灯发出暖黄色光芒，
中景：床铺整洁，枕头柔软，
远景：窗帘半掩，月光透过，
暖黄色调，温馨舒适，
画面干净无文字无logo，16:9
```

**场景2：办公室（午休补涂）**：
```
电商产品摄影风格，画面中不出现任何角色；纯场景、纯环境、无人物，
中景视角，平视，
室内，办公室，午后，
前景：白色办公桌，笔记本电脑，咖啡杯，
中景：文件整齐摆放，绿植点缀，
远景：落地窗，阳光透入，
明亮清爽，专业简洁，
画面干净无文字无logo，16:9
```

**入库命令**：
```bash
python gen-video/scripts/update_material_files.py "./素材库.json" \
  --scene "卧室" "https://场景1URL/" "夜晚"
```

### 第三步：出镜人物（可选）

**核心规则**：如果视频需要人物出镜，在产品和场景之后生成。

**示例**：

## 🎨 产品-场景融合四要素

**核心规则**：产品图和场景图融合时，必须保证物理逻辑自洽。

| 要素 | 说明 | 写法示例 |
|------|------|---------|
| **光照一致** | 产品和场景的光源方向相同 | `柔光从左上方打来，瓶身左侧有高光过渡到右侧阴影` |
| **色温协调** | 产品的高光/阴影色温和环境匹配 | `瓶身玻璃折射出周围环境的暖黄色调` |
| **物理接触** | 投影、反射、接触面 | `台面上投射出清晰的瓶身投影` |
| **材质互动** | 产品材质对环境光的响应 | `金属表面映出窗外秋叶的模糊倒影` |

**示例 Prompt（产品融入场景）**：
```
0-4s：【中景】场景[图片2]（卧室），产品[图片1]放在床头柜上，台灯暖光从左上方打来，瓶身左侧有高光，右侧有柔和阴影，台面上投射出瓶身的清晰投影。旁白说："睡前使用，第二天肌肤水润饱满。"
不添加字幕，不添加BGM
```

## 💬 旁白处理

**核心规则**：电商广告的旁白是画外音（VO），V3 阶段必须创建旁白虚拟角色。

**原因**：
- 电商广告通常不需要人物口播
- 旁白作为画外音，可以聚焦产品画面
- V5 阶段用 `旁白说："台词"` 格式

**V3 阶段入库**：
```bash
python gen-video/scripts/update_material_files.py "./素材库.json" \
  --narrator "https://s.coze.cn/image/kgWvY_NvVcQ/" \
  --voice "女声，活泼热情，语速明快" \
  --tts \
  --voice-name "咪仔"
```

**V5 阶段使用**：
```
旁白说："这款面霜，让你的肌肤焕发新生。"
```

## 🎵 BGM 配乐建议

**核心规则**：电商视频推荐添加 BGM，风格轻快明亮。

**BGM 风格推荐**：
- 轻快流行：吉他 + 手拍鼓
- 明亮电子：合成器 + 轻鼓点
- 温暖民谣：木吉他 + 口琴

**示例调用**：
```python
select_music(
    music_requirements="轻快明亮的纯音乐，适合产品展示",
    video_script_file="面霜广告_视频脚本.cvs"
)
```

## ⏱️ 时长策略

**核心规则**：优先用长 shot + 分时序，不要拆短 shot。

**原因**：
- 电商视频注重产品质感展示
- 长镜头可以充分展示产品细节
- 分时序可以控制节奏和信息密度

**示例**：
```
❌ 错误做法：
Shot 1（3s）：产品特写
Shot 2（3s）：产品中景
Shot 3（3s）：产品全景

✅ 正确做法：
Shot 1（9s）：
0-3s：【特写】产品[图片1]在柔光下旋转。
3-6s：【中景】产品放在台面上，背景虚化。
6-9s：【全景】产品居中，场景全貌。
```

## ⛔ 常见错误与规避

### 错误1：产品与场景一次性生成

❌ **错误做法**：
```
产品放在卧室床头柜上的场景图
```

✅ **正确做法**：
```
# 第一步：生成产品图（纯白背景）
磨砂玻璃瓶身，乳白色液体，金色瓶盖，纯白背景

# 第二步：生成场景图（无产品）
卧室场景，木质床头柜，台灯暖光，无产品

# 第三步：V5 分镜融合
产品[图片1]放在场景[图片2]的床头柜上
```

**原因**：产品与场景分离可以灵活调整产品位置、光照、角度。

---

### 错误2：忽略光照一致性

❌ **错误做法**：
```
场景是午后阳光从右侧透入，产品光照从左上方打来
```

✅ **正确做法**：
```
场景是午后阳光从右侧透入，产品光照也从右侧打来，瓶身右侧有高光
```

**原因**：光照不一致会破坏画面的真实感。

---

### 错误3：旁白与画面不同步

❌ **错误示例**：
```
0-4s：【特写】产品旋转。旁白说："睡前使用，第二天肌肤水润饱满。"
```

✅ **正确示例**：
```
0-4s：【中景】产品放在床头柜上（卧室场景）。旁白说："睡前使用，第二天肌肤水润饱满。"
```

**原因**：旁白提到"睡前"，画面应该是卧室场景，而不是产品旋转特写。
