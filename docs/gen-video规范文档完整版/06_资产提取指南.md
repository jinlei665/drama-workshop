# 资产提取执行手册

> **所属 Skill**: gen-video
> **核心用途**: 规范从剧本/分镜规划到素材生成的完整资产提取流程
> **何时读取**: V3 阶段生成参考图时必读
> **前置确认**: 用户已确认视觉风格（V2 完成）

---

## 🎯 核心原则

**资产分离铁律**：
- 角色图 = 纯白背景 + 角色本体（不含环境、不含手持物）
- 场景图 = 纯场景 + 纯环境（不含人物）
- 道具图 = 纯白背景 + 道具本体

**为什么分离？** V5 视频生成引擎需要独立的资产进行动态融合。角色带环境、场景带人物会导致融合冲突。

### 第四步：生成资产参考图

#### 4.1 角色参考图生成

**工具选择**：
- **text_to_images**：无参考图，从零生成
- **image_to_images**：有参考图，基于参考图调整

**Prompt 公式（遵循 `prompt-writing-guide.md`）**：

```
16:9 横版构图，[unified_style_prompt]。
左半边区域居中展示角色面部特写，
右半边区域整齐排布角色正面、侧面及背面三张全身设定图。
纯白背景，无文字无水印。
[性别+年龄段+身高+体型]，[面孔特征]，
[整体气质关键词，日常神态]，
[发色+发长+发式+质感]，[脸型+皮肤+五官特征]，
[服装款式+色彩+材质]，[配饰+鞋靴]，
[站姿描述]
```

**示例**：
```
16:9 横版构图，中国水墨动画风格，传统水墨晕染笔触，黑白灰为主调偶用朱砂点缀。
左半边面部特写，右半边正面/侧面/背面三张全身设定图。
纯白背景，无文字无水印。
男，二十岁左右，中等身高，体型修长，东方人面孔，
气质温和内敛，眉眼柔和，常带浅笑，
黑色长发束起，墨色中衣，白色外袍，腰系玉佩，
站姿自然，双手垂于身侧。
```

**多景别处理**：
- **同一角色，不同景别**：使用 `image_to_images` 基于已有设定图生成
- **不同视角（正面/侧面/背面）**：初始 Prompt 已包含三视图，一次生成即可

#### 4.2 场景参考图生成

**Prompt 公式**：

```
[unified_style_prompt]，
画面中不出现任何角色；纯场景、纯环境、无人物，
[视角：从哪里看向哪里+景别]，
[空间类型：室内/室外+时代/世界观]，
[前景：距镜头最近的物体]，
[中景：场景主体结构]，
[远景：天际线/背景/纵深尽头]，
[关键物件：有叙事功能的特定物体]，
[光照：光源方向+类型+强度]，
[色调与氛围：主色调+情绪关键词]，
画面干净无文字无logo，[比例]
```

**场景多景别生成方法**：

| 追加景别 | `image_to_images` Prompt 示例 |
|----------|------|
| **侧面视角** | `将视角修改为[核心元素]的侧面视角，[元素A]与[元素B]沿纵深排列。不再展示[正面才可见的元素]。保持画面风格，无人物` |
| **反向视图** | `将视角反转180度，从[原背景方向]看向[原前景方向]。不再展示[原背景元素]。保持画面风格，无人物` |
| **同光照变体** | `保持相同视角构图，将光照改为[夜晚月光/黄昏暖光等]。保持画面风格，无人物` |

**示例（同一场景，多景别）**：

1. **全景**：
```
中国水墨动画风格，画面中不出现任何角色；纯场景、纯环境、无人物，
全景视角，从高处俯瞰小镇，
室外，古代小镇，春日清晨，
前景：小溪潺潺，桃花枝条探入画面，
中景：白墙黑瓦的民居沿溪而建，炊烟袅袅，
远景：远山如黛，竹林掩映，
晨雾笼罩，柔光弥漫，
青绿色调为主，偶有桃花粉色点缀，宁静治愈，
画面干净无文字无logo，16:9
```

2. **中景（基于全景生成）**：
```
将视角推进到小镇中心街道，两侧是白墙黑瓦的民居，中间是青石板路。
保留晨雾和桃花元素。
保持画面风格，无人物。
```

3. **反向视图（从中景生成）**：
```
将视角反转180度，从街道看向小溪方向。
不再展示街道两侧的民居，展示溪流和对岸的竹林。
保持画面风格，无人物。
```

#### 4.3 道具参考图生成

**Prompt 公式**：

```
[unified_style_prompt]，
[物件类型+名称]，纯白背景，正面视角，
[材质+表面质感]，[形状+颜色+尺寸参照]，[细节/痕迹]，
光照均匀，无文字无水印，[比例]
```

**示例**：
```
中国水墨动画风格，
古旧玉佩，纯白背景，正面视角，
青白玉材质，表面有岁月磨损的细腻纹理，
圆形，直径约掌心大小，中央雕刻云纹，边缘有细小缺口，
光照均匀，无文字无水印，16:9
```

### 第五步：资产安全入库

**入库命令**：

```bash
# 角色入库（管线 A - 漫剧/氛围片/预告片）
python gen-video/scripts/update_material_files.py "./素材库.json" \
  --role "陈念" "https://s.coze.cn/image/xxx/" "正面" \
  --voice "男声，青年音色，温和清亮，语速中等，带有书卷气"

# 角色入库（管线 B - 数字人/科普/电商）
python gen-video/scripts/update_material_files.py "./素材库.json" \
  --role "讲解员" "https://s.coze.cn/image/xxx/" "正面" \
  --voice "女声，专业清亮，语速明快" \
  --tts

# 场景入库
python gen-video/scripts/update_material_files.py "./素材库.json" \
  --scene "青溪镇春晨" "https://s.coze.cn/image/xxx/" "全景"

# 道具入库
python gen-video/scripts/update_material_files.py "./素材库.json" \
  --item "古旧玉佩" "https://s.coze.cn/image/xxx/" "特写"

# 旁白入库（需要旁白时）
python gen-video/scripts/update_material_files.py "./素材库.json" \
  --narrator "https://s.coze.cn/image/kgWvY_NvVcQ/" \
  --voice "男声，老年音色，沉稳有力，带有故事感"
```

**⚠️ 同一资产多文件处理**：
- 同一角色/场景的不同景别图，分别入库，备注中注明景别
- 示例：
```bash
# 陈念 - 正面全身
python gen-video/scripts/update_material_files.py "./素材库.json" \
  --role "陈念" "https://url1/" "正面全身"

# 陈念 - 侧面
python gen-video/scripts/update_material_files.py "./素材库.json" \
  --role "陈念" "https://url2/" "侧面"
```

## ⛔ 常见错误与规避

### 错误1：角色图带环境

❌ **错误做法**：
```
陈念站在小屋门口，身后是木质门框，晨光从侧面打来
```

✅ **正确做法**：
```
陈念，纯白背景，正面全身设定图，不画环境
```

**原因**：角色图带环境会导致 V5 融合时环境冲突。

---

### 错误2：场景图带人物

❌ **错误做法**：
```
小镇街道，陈念从远处走来
```

✅ **正确做法**：
```
小镇街道，纯场景，无人物
```

**原因**：场景图中的人物在 V5 融合时会与角色图冲突。

---

### 错误3：忽略景别差异

❌ **错误做法**：
- 只生成一张场景全景图
- V5 写分镜时需要中景/近景时，直接用全景图引用

✅ **正确做法**：
- 规划阶段列出景别需求
- 为每个关键景别生成独立的参考图
- V5 引用时选择景别匹配的图

**原因**：全景图推近景会导致画面模糊，失去细节。

---

### 错误4：道具与角色不分离

❌ **错误做法**：
```
陈念手持玉佩，玉佩悬在空中
```

✅ **正确做法**：
- 角色图：陈念，纯白背景，双手自然下垂（不拿东西）
- 道具图：玉佩，纯白背景，正面特写
- V5 分镜 Prompt：`陈念[图片1]伸出右手，掌心托起玉佩[图片2]`

**原因**：角色手持道具的参考图会限制 V5 的动作自由度。
