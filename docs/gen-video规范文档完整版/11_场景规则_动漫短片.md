# 场景规则：动漫短片 (anime_short)

> **所属 Skill**: gen-video
> **scene-rule ID**: anime_short
> **核心策略**: 稳定性优先，简化景别
> **适用场景**: 漫剧、二次元短剧、动画短片

---

## 🎯 核心特征

**动漫短片的核心挑战是 AI 视频生成的稳定性**。多角色、复杂动作、频繁运镜都可能导致画面崩坏。因此，动漫短片的规则核心是"稳定性优先"——通过固定镜头、简化景别、分时序控制来实现可控输出。

---

## 📋 规则详情

| 维度 | 规则 |
|------|------|
| **稳定性** | 多采用固定镜头，减少运镜幅度，避免画面抖动和角色崩坏 |
| **景别简化** | 优先利用长镜头分时序实现景别变化。避免一个画面内同时出现 3 个以上角色 |
| **正反打处理** | 对话回合必须在同一 shot_id 内用分时序交替，禁止拆成多个 shot |
| **分时序密度** | 对话场景：3-4s/段；动作场景：2-3s/段；空镜/过渡：4-5s/段 |
| **场景规划** | 每分钟视频至少 2 个不同场景，色调应有明暗或冷暖对比 |
| **旁白处理** | 两种模式：① 角色自述型（用角色名说）② 第三人称旁白型（创建旁白虚拟角色） |
| **BGM** | ❌ 不加 |
| **音色管线** | 🅰️ **管线 A（描述音色）** |

---

## 🎬 分镜设计要点

### 1. 固定镜头优先

**运镜建议**：
- 默认：固定镜头（镜头静止，画面稳定）
- 可选：缓慢横向移动（轨道镜头）
- 避免：快速推进/拉远、大幅度的手持晃动

**原因**：AI 视频生成中，运镜幅度越大，画面稳定性越差。固定镜头是保证画面质量的首选。

**示例 Prompt**：
```
0-4s：【中景】镜头固定。角色[图片1]和角色[图片2]面对面站立，相距两步。
4-8s：【近景】镜头切到角色[图片1]的面部，她的眼神坚定。
8-12s：【近景】镜头切到角色[图片2]的面部，他微微皱眉。
不添加字幕，不添加BGM
```

### 2. 景别简化策略

**核心原则**：一个 shot 内通过分时序实现景别变化，而不是频繁切镜。

**景别变化方法**：
- **全景 → 中景**：分时序第一段全景，第二段中景
- **中景 → 近景**：分时序第一段中景，第二段近景
- **正反打对话**：同一 shot 内用分时序交替切镜头

**示例（正反打对话）**：
```
0-3s：【中景】角色[图片1]站在左侧，面向右侧。角色[图片1]说："你真的决定了吗？"
3-6s：【中景】角色[图片2]站在右侧，面向左侧，点头。角色[图片2]说："我已经想好了。"
6-9s：【近景】镜头切到角色[图片1]的面部，她的眼神中带着担忧。
9-12s：【近景】镜头切到角色[图片2]的面部，他轻轻握拳，眼神坚定。角色[图片2]说："我会保护她的。"
不添加字幕，不添加BGM
```

### 3. 多角色控制

**核心规则**：一个画面内同时出现的角色 ≤ 3 个。

**原因**：AI 视频生成中，画面内角色越多，角色崩坏的概率越高。

**应对策略**：
- **分组对话**：将多角色场景拆成多个 shot，每个 shot 聚焦 2-3 个角色
- **分时序聚焦**：同一 shot 内用分时序切换焦点角色，而不是让所有角色同时出现
- **背景虚化**：次要角色用虚化背景处理，不精细刻画

**示例（4 角色对话场景）**：
```
Shot 1（聚焦 A 和 B）：
0-4s：【中景】角色[图片1]和角色[图片2]面对面站立。角色[图片1]说："我们分头行动。"
4-8s：【近景】角色[图片2]点头。角色[图片2]说："明白。"

Shot 2（聚焦 C 和 D）：
0-4s：【中景】在背景中，角色[图片3]和角色[图片4]已经在远处等待。角色[图片3]说："准备好了。"
4-8s：【全景】四人一起向前走去。
不添加字幕，不添加BGM
```

## 💬 旁白处理模式

### 模式 1：角色自述型

**适用场景**：第一人称视角的内心独白。

**格式**：
```
角色名说："台词"
```

**示例**：
```
0-4s：【近景】角色[图片1]独自站在窗前，望着窗外的夜空。
4-8s：【特写】镜头推近角色的侧脸，眼神中带着回忆。角色[图片1]说："从那天起，我就一直在寻找答案。"
不添加字幕，不添加BGM
```

### 模式 2：第三人称旁白型

**适用场景**：全知视角的叙事旁白。

**前置条件**：V3 阶段创建"旁白"虚拟角色入库。

**格式**：
```
旁白说："台词"
```

**示例**：
```
0-5s：【全景】场景[图片1]的城市在夜幕中灯火通明。
5-10s：【中景】镜头缓慢推进，聚焦在街道上的行人。旁白说："这座城市，每个人都在寻找自己的归宿。"
不添加字幕，不添加BGM
```

## 🎨 场景规划建议

**核心规则**：每分钟视频至少 2 个不同场景，色调应有明暗或冷暖对比。

**原因**：
- 多场景切换可以丰富视觉层次
- 色调对比可以增强情绪张力

**场景规划示例（2 分钟漫剧）**：

| Shot | 场景 | 时长 | 色调 | 情绪 |
|------|------|------|------|------|
| 1 | 雨夜街道（外景） | 12s | 冷蓝 | 忧郁 |
| 2 | 咖啡店（内景） | 15s | 暖黄 | 温暖 |
| 3 | 雨夜街道（外景） | 10s | 冷蓝 | 忧郁 |
| 4 | 公园长椅（外景） | 8s | 暖橙 | 希望 |

---

## ⏱️ 分时序密度建议

| 场景类型 | 建议时长/段 | 原因 |
|---------|-----------|------|
| 对话场景 | 3-4s | 对话节奏较快，台词简洁 |
| 动作场景 | 2-3s | 动作流畅，节奏紧凑 |
| 空镜/过渡 | 4-5s | 给观众情绪过渡时间 |

**示例（对话场景）**：
```
0-3s：【中景】角色[图片1]皱眉。角色[图片1]说："这不对劲。"
3-6s：【近景】角色[图片2]的眼神闪烁。角色[图片2]说："也许我们误会了。"
6-9s：【中景】两人对视，气氛紧张。
不添加字幕，不添加BGM
```

**示例（动作场景）**：
```
0-2s：【全景】角色[图片1]从高处跃下。
2-4s：【中景】角色落地，膝盖微曲缓冲。
4-6s：【近景】角色抬头，眼神锐利。
6-8s：【全景】角色快速向前奔跑，消失在巷道尽头。
不添加字幕，不添加BGM
```

## 📝 完整示例

### 场景：雨夜重逢

**剧本**：
- 男主在雨夜街道上等待，女主撑伞走来。
- 两人简短对话，男主决定保护女主。
- 情绪：忧郁、温暖、坚定

**分镜设计**：

**Shot 1**（12s）：
```
0-4s：【全景】雨夜的街道，街灯投下昏黄的光。角色[图片1]独自站在街角，雨水打湿了他的头发。
4-8s：【中景】镜头缓慢推进，角色的眼神望向街道尽头，等待。
8-12s：【近景】角色的嘴角轻轻抿起，眼神中带着一丝期待。
不添加字幕，不添加BGM
```

**Shot 2**（15s）：
```
0-3s：【全景】远处，一把红色雨伞在雨幕中出现。角色[图片2]撑伞缓缓走来。
3-6s：【中景】角色[图片1]向前迈出一步，眼神锁定在角色[图片2]身上。
6-9s：【中景】角色[图片2]停下脚步，两人相距三步。角色[图片2]说："好久不见。"
9-12s：【近景】角色[图片1]的眼神柔和下来。角色[图片1]说："你终于来了。"
12-15s：【全景】两人并肩站在雨中，红色雨伞遮住了两人的身影。
不添加字幕，不添加BGM
```

## ⛔ 常见错误与规避

### 错误1：对话拆成多个 shot

❌ **错误做法**：
```
Shot 1：角色A说："你决定了吗？"
Shot 2：角色B说："是的。"
Shot 3：角色A说："那我陪你。"
```

✅ **正确做法**：
```
Shot 1：
0-3s：【中景】角色[图片1]说："你决定了吗？"
3-6s：【中景】角色[图片2]点头。角色[图片2]说："是的。"
6-9s：【近景】角色[图片1]的眼神坚定。角色[图片1]说："那我陪你。"
不添加字幕，不添加BGM
```

**原因**：频繁切 shot 会导致画面不连贯，分时序在同一 shot 内切换更自然。

---

### 错误2：一个画面 4+ 角色

❌ **错误做法**：
```
【全景】四个角色围坐在桌前，进行激烈讨论。
```

✅ **正确做法**：
```
Shot 1：
0-4s：【中景】角色[图片1]和角色[图片2]面对面。角色[图片1]说："我们分头行动。"

Shot 2：
0-4s：【中景】在背景中，角色[图片3]和角色[图片4]已经在远处等待。角色[图片3]说："准备好了。"
不添加字幕，不添加BGM
```

**原因**：AI 视频生成中，画面内角色越多，崩坏概率越高。

---

### 错误3：运镜幅度过大

❌ **错误做法**：
```
0-5s：【全景→特写】镜头快速推进，从全景直切到面部特写。
```

✅ **正确做法**：
```
0-5s：【全景】镜头固定，画面稳定。
5-10s：【中景】镜头缓慢推进，节奏舒缓。
10-15s：【近景】镜头继续缓慢推进，聚焦面部。
不添加字幕，不添加BGM
```

**原因**：大幅度快速运镜容易导致画面抖动和角色崩坏。

## 📋 检查清单

| # | 检查项 |
|---|--------|
| 1 | 运镜以固定镜头为主，避免大幅度运镜 |
| 2 | 一个画面内同时出现的角色 ≤ 3 个 |
| 3 | 对话回合在同一 shot 内用分时序交替 |
| 4 | 分时序密度：对话 3-4s/段，动作 2-3s/段 |
| 5 | 每分钟至少 2 个不同场景 |
| 6 | 场景之间有色调对比（明暗/冷暖） |
| 7 | 旁白格式正确（角色名说/旁白说） |
| 8 | 音色管线使用管线 A（描述音色） |
