## 一、统一风格提示词模板

**作用**：贯穿全片所有生图环节的视觉基调，确保角色图、场景图、视频分镜风格一致。

### 1.1 获取方式

调用风格查询脚本获取完整提示词：
```bash
python gen-video/scripts/get_style_prompt.py [风格编号] --full
```

### 1.2 通用格式

```
[风格核心技术词]，[笔触/渲染技法]，[主色调+点缀色]，[材质质感]，[人物/场景特征]，[氛围渲染]，[美学流派/参考]
```

### 1.3 常用风格示例

| 风格编号 | 风格名称 | 完整提示词 |
|---------|---------|-----------|
| 11 | 2D国风 | 中国传统动画风格，赛璐璐着色，柔和的东方美学色调，流畅的线条勾勒，唯美古风意境，仙侠浪漫氛围 |
| 16 | 2D水墨武侠 | 中国水墨动画风格，传统水墨晕染笔触，飞白泼墨技法，黑白灰为主调偶用朱砂/石青点缀，宣纸纹理质感，人物轮廓写意灵动，云雾淡墨渲染虚实相生，上海美术电影制片厂美学 |
| 19 | 3D国风 | 3D写实渲染，次世代PBR材质，东方玄幻美术风格，仙侠光影特效，精致建模细节，电影级构图，虚幻引擎5质感 |
| 23 | 吉卜力 | 吉卜力工作室风格，手绘水彩质感，柔和自然光线，清新治愈色调，细腻背景绘制，温馨日常氛围 |

### 1.4 使用规则

- **必须作为开头**：所有 V3/V5 阶段的提示词必须以统一风格提示词开头
- **不可混用**：同一视频项目只能选择一种风格，不可中途更换
- **保持完整**：从脚本获取的提示词需完整保留，不可删减核心词汇

## 二、角色三视图设定稿模板

**作用**：从文本描述生成角色参考图，建立人物视觉锚点。

### 2.1 核心原则

- **角色与背景分离**：纯白背景，不带环境，不带手持道具
- **结构化描述**：按槽位顺序填充，不可跳过必填项
- **冗余加固**：关键特征需重复强调（如"面向镜头，五官清晰可见"）

### 2.2 通用公式

```
16:9 横版构图，[unified_style_prompt]。
左半边区域居中展示角色面部特写，
右半边区域整齐排布角色正面、侧面及背面三张全身设定图。
纯白背景，无文字无水印。
[①体型：性别+年龄段+身高+体型]，[②面孔：东方人/西方人面孔]，
[③气质：整体气质关键词，日常神态]，
[④发型：发色+发长+发式+质感]，
[⑤面部：脸型+皮肤+五官特征]，
[⑥服装主体：款式+色彩+材质]，
[⑦服装细节：配饰+鞋靴]，
[⑧身体标记：伤疤/纹身/胎记（如有）]，
[⑨姿态：站姿描述]
```

### 2.3 槽位详细说明

| 槽位 | 必填 | 提取来源 | 示例 |
|------|------|---------|------|
| ①体型 | ✅ | 文本中的性别、年龄、身材描述 | 男性，二十岁左右，修长身形 |
| ②面孔 | ✅ | 文本中的民族/种族特征 | 东方人面孔 |
| ③气质 | ✅ | 文本中的性格、神态关键词 | 气质内敛文雅，日常神态略带忧郁 |
| ④发型 | ✅ | 文本中的发型描述 | 黑色长发束于脑后，额头有碎发 |
| ⑤面部 | ✅ | 文本中的五官描述 | 脸型清秀，皮肤白皙，五官端正 |
| ⑥服装主体 | ✅ | 文本中的服装描述 | 白色中衣，粗布质地，宽松款式 |
| ⑦服装细节 | ⚪ | 文本中的配饰描述 | 腰间系带，黑色布鞋 |
| ⑧身体标记 | ⚪ | 文本中的伤疤/纹身描述 | 无伤疤纹身（如无则填"无"或省略） |
| ⑨姿态 | ⚪ | 默认站姿 | 站姿自然放松，双手垂于身侧，面向镜头，五官清晰可见 |

### 2.4 提取示例

**原文**：

**填充后**：
```

右半边区域整齐排布角色正面、侧面及背面三张全身设定图。
```

## 三、场景参考图模板

**作用**：从文本描述生成场景参考图，建立空间视觉锚点。

### 3.1 核心原则

- **场景与人物分离**：纯场景，不带人物
- **分层描述**：前景→中景→远景，逐层构建空间
- **光影明确**：光源方向、类型、强度必须具体

### 3.2 通用公式

```
[unified_style_prompt]，
画面中不出现任何角色；纯场景、纯环境、无人物，
[①视角：从哪里看向哪里+景别]，
[②空间类型：室内/室外+时代/世界观]，
[③前景：距镜头最近的物体]，
[④中景：场景主体结构]，
[⑤远景：天际线/背景/纵深尽头]，
[⑥关键物件：有叙事功能的特定物体]，
[⑦光照：光源方向+类型+强度]，
[⑧色调与氛围：主色调+情绪关键词]，
画面干净无文字无logo，[比例]
```

### 3.3 槽位详细说明

| 槽位 | 必填 | 提取来源 | 示例 |
|------|------|---------|------|
| ①视角 | ✅ | 文本中的镜头位置 | 从高处俯瞰古风小镇全景，远景构图 |
| ②空间类型 | ✅ | 文本中的地点描述 | 古风小镇春晨，室外，古代 |
| ③前景 | ⚪ | 文本中距镜头近的元素 | 青石板街道两侧白墙黑瓦的民居 |
| ④中景 | ✅ | 文本中的场景主体 | 小镇街巷错落有致，柳树抽出嫩芽 |
| ⑤远景 | ⚪ | 文本中远处的元素 | 黛青色山峦连绵起伏，山脚竹林 |
| ⑥关键物件 | ⚪ | 文本中有叙事功能的物品 | 炊烟从烟囱袅袅升起 |
| ⑦光照 | ✅ | 文本中的光线描述 | 晨曦微亮，淡金色阳光从山后透出 |
| ⑧色调与氛围 | ✅ | 文本中的情绪词 | 宁静古朴治愈的春天清晨朦胧感 |

## 四、分镜视频提示词模板

**作用**：生成视频的核心指令，将角色放入场景并赋予动作。

### 4.1 核心原则

- **素材引用锚定**：prompt 中出现的角色/场景必须挂载 `[图片N]`
- **物理动作具象化**：禁止抽象动词（沉思、激烈战斗），必须转化为可见动作
- **分时序切分**：每个 shot 必须使用时间区间标记，每段 3-5 秒
- **防幻觉结尾**：每个 prompt 末尾必须加 `不添加字幕，不添加BGM`

### 4.2 无台词场景公式

```
[时间区间]：【景别】[场景名][图片N]内，[环境光影描述]。
[角色名][图片N] [具体动作]，[微表情变化]，[肢体语言]。
[环境细节补充]。
[运镜指令]，[光影/物理变化]。
不添加字幕，不添加BGM
```

### 4.3 有台词场景公式

```
[时间区间]：【景别】[场景名][图片N]内，[环境描述]。
[角色名][图片N] [动作描述]。[角色名]说："台词"。
不添加字幕，不添加BGM
```

### 4.4 多段分时序公式

```
[时间区间1]：【景别1】[场景名][图片N]内，[动作/环境描述]。
[时间区间2]：【景别2】[角色名][图片N] [动作变化]。[角色名]说："台词"。
[时间区间3]：【景别3】[角色名][图片N] [情绪变化]，[动作]。
不添加字幕，不添加BGM
```

### 4.5 关键要素说明

| 要素 | 必填 | 说明 | 示例 |
|------|------|------|------|
| 时间区间 | ✅ | 格式：`0-5s` / `5-10s` | 0-5s |
| 景别 | ✅ | 全景/中景/近景/特写 | 【全景】【中景】 |
| 素材引用 | ✅ | 角色和场景必须挂载 | 古风小镇[图片1]，陈念[图片2] |
| 动作描述 | ✅ | 具象化物理动作 | 缓缓抬眼望向窗外 |
| 运镜指令 | ⚪ | 固定/推/拉/移/摇 | 镜头缓慢推远 |
| 台词 | ⚪ | 格式：角色名说："台词" | 陈念说："今天天气真好。" |

### 4.6 时长计算规则

**有台词时**：
```
时长 = 台词字数 × 300ms + 500ms 缓冲
向上取整到整千数
```

**示例**：
- 10字台词 → 10 × 300 + 500 = 3500ms → 4000ms
- 20字台词 → 20 × 300 + 500 = 6500ms → 7000ms

**无台词时**：
```
简单动作/空镜：4000-6000ms
复杂动作链：8000-12000ms
单 shot 上限：15000ms
```

### 4.7 提取示例

**原文**：
> 分镜2：陈念醒来·晨光透窗。时长：5秒。简朴木质小屋，陈念身穿白色中衣坐在床沿发呆，晨光透过窗纸洒入，固定镜头。

**填充后**：
```
0-5s：【中景】简朴木质小屋[图片1]内，晨光透过窗纸洒入，淡金色光线照亮半个房间。
陈念[图片2]身穿白色中衣坐在床沿，双手放于膝上，目光微微低垂，眉眼温和，
发呆片刻后缓缓抬眼望向窗外。
屋顶横梁，木质货柜靠墙，墙角脸盆架，铜镜置于桌上。
固定镜头，晨光缓慢移动。
不添加字幕，不添加BGM
```

## 五、文本解析提取策略

当用户提供剧本/分镜描述时，按以下优先级提取信息。

### 5.1 角色提取槽位

| 槽位 | 提取关键词 | 示例 |
|------|-----------|------|
| 基础信息 | 姓名、性别、年龄 | 陈念、男性、二十岁左右 |
| 外貌特征 | 发色、发型、五官、脸型、肤色 | 黑色长发、眉眼温和、脸型清秀 |
| 服装 | 款式、颜色、材质、配饰 | 白色中衣、粗布质地、腰间系带 |
| 气质 | 性格关键词、日常神态 | 内敛文雅、略带忧郁 |
| 身体标记 | 伤疤、纹身、胎记 | 左臂有烧伤疤痕 |

### 5.2 场景提取槽位

| 槽位 | 提取关键词 | 示例 |
|------|-----------|------|
| 空间类型 | 室内/室外、时代背景 | 室内、古代、木质小屋 |
| 景别 | 全景/中景/近景/特写 | 远景构图 |
| 构图层次 | 前景、中景、远景 | 前景街道、中景民居、远山 |
| 光源 | 方向、类型、强度 | 晨光从左侧窗户射入 |
| 氛围 | 情绪关键词 | 宁静、古朴、治愈、孤独感 |
| 关键物件 | 有叙事功能的物品 | 铜镜、货柜、脸盆架 |

### 5.3 动作提取槽位

| 槽位 | 提取关键词 | 示例 |
|------|-----------|------|
| 时序 | 动作先后顺序 | 发呆 → 抬眼望向窗外 |
| 物理动作 | 具象化描述 | 双手放于膝上、缓缓抬眼 |
| 微表情 | 眼神、面部变化 | 目光低垂、眉眼温和 |
| 运镜 | 固定/推/拉/移/摇 | 固定镜头、缓慢推远 |
| 光影变化 | 光线移动、明暗变化 | 晨光缓慢移动 |
