# 工作流编辑器 - 使用说明

## 访问方式

### 方式 1：侧边栏导航
点击左侧侧边栏的「工作流」菜单项

### 方式 2：首页入口
1. 访问首页：http://localhost:5000
2. 点击「工作流编辑器」卡片

### 方式 3：直接访问
访问 http://localhost:5000/workflow

## 核心功能

### 1. 节点库（左侧面板）

提供以下节点类型：

- **文本输入** - 输入文本内容
- **图片输入** - 输入图片 URL
- **文生图** - 文本生成图片
- **图生图** - 图片修改生成
- **图生视频** - 图片生成视频
- **文生视频** - 文本生成视频
- **LLM 处理** - 大语言模型处理

### 2. 画布操作（中间区域）

- **拖拽添加节点**：从左侧节点库拖拽节点到画布
- **移动节点**：按住节点左键拖拽移动位置
- **连接节点**：
  1. 点击节点的输出端口（右侧圆点）
  2. 拖拽到另一个节点的输入端口（左侧圆点）
  3. 松开鼠标完成连接
- **删除连接**：点击连接线即可删除
- **删除节点**：点击节点右上角删除按钮
- **缩放画布**：使用鼠标滚轮缩放
- **平移画布**：按住 Alt 键 + 鼠标左键拖拽

### 3. 参数面板（右侧 - 参数标签）

点击任意节点，右侧参数面板会显示该节点的所有可配置参数：

**文生图节点参数**：
- 提示词（多行文本）
- 负面提示词（多行文本）
- 宽度（下拉选择：512/768/1024/1360）
- 高度（下拉选择：512/768/1024/1360）
- 风格（下拉选择：realistic/anime/cartoon/oil_painting/watercolor）
- 随机种子（数字：-1 为随机）
- 迭代步数（滑块：10-50）
- 引导强度（滑块：1-20）

**图生图节点参数**：
- 修改提示词（多行文本）
- 修改强度（滑块：0-1）
- 随机种子
- 迭代步数
- 引导强度

**图生视频节点参数**：
- 动作描述（多行文本）
- 时长（滑块：4-12秒）
- 比例（下拉选择：16:9/9:16/1:1）
- 运动强度（滑块：1-10）

**文生视频节点参数**：
- 场景描述（多行文本）
- 时长（滑块：4-12秒）
- 比例（下拉选择：16:9/9:16/1:1）

### 4. 预览面板（右侧 - 预览标签）

执行工作流后，在预览面板查看结果：
- 图片预览（生成的图片）
- 视频预览（生成的视频）
- 文本输出（LLM 处理结果）

### 5. 工作流执行

- 点击顶部「执行工作流」按钮
- 自动根据节点依赖关系执行
- 支持并行执行无依赖的节点
- 节点状态实时显示（运行中/成功/失败）

## 使用示例

### 示例 1：文生图工作流

1. 从左侧拖拽「文本输入」节点到画布
2. 在参数面板的文本内容中输入：`一只可爱的橘猫坐在窗边，阳光洒进来，写实风格`
3. 从左侧拖拽「文生图」节点到画布
4. 点击「文本输入」节点的右侧端口（text），拖拽到「文生图」节点的左侧端口（prompt）
5. 在「文生图」节点参数中设置：
   - 宽度：1024
   - 高度：1024
   - 风格：realistic
   - 迭代步数：30
6. 点击顶部「执行工作流」按钮
7. 在右侧「预览」标签查看生成的图片

### 示例 2：图生视频工作流

1. 拖拽「图片输入」节点到画布
2. 在参数面板输入图片 URL（可以使用 oss:// 开头的图片地址）
3. 拖拽「图生视频」节点到画布
4. 连接「图片输入」的 image 输出到「图生视频」的 referenceImage 输入
5. 在「图生视频」节点参数中设置：
   - 动作描述：`猫咪缓慢地转过头看向镜头，柔和的光线变化`
   - 时长：5秒
   - 比例：16:9
   - 运动强度：5
6. 点击「执行工作流」按钮
7. 在预览面板查看生成的视频

### 示例 3：复杂工作流

```
文本输入 → LLM 处理 → 文生图 → 图生视频
```

1. 添加「文本输入」节点，输入原始描述
2. 添加「LLM 处理」节点，设置处理指令优化描述
3. 添加「文生图」节点，生成图片
4. 添加「图生视频」节点，生成视频
5. 按顺序连接所有节点
6. 执行工作流

## 故障排除

### 问题：节点无法连接
- 确保点击的是输出端口（右侧圆点）
- 拖拽到输入端口（左侧圆点）
- 松开鼠标完成连接

### 问题：参数面板不显示
- 确保选中了节点（节点周围有高亮边框）
- 点击节点即可选中

### 问题：执行失败
- 检查节点是否正确连接
- 检查参数是否填写完整
- 查看节点状态（红色表示失败）

### 问题：预览不显示结果
- 确保工作流已执行
- 选中节点查看预览
- 检查节点执行状态
